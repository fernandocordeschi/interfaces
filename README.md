# 🎮 TP Interfaces – Proyecto Final

## 📄 Descripción
Este proyecto es el Trabajo Práctico final de la materia de Interfaces.  
Se trata de una **aplicación web interactiva** que incluye:  

- 🔑 Pantalla de **login** y **registro** de usuarios.  
- 🏠 Pantalla de **inicio** con un **carousel principal** y varios **carousels por categorías**.  
- 🕹️ Juegos interactivos, incluyendo un clon de **Flappy Bird**.  
- ✨ Navegación sencilla y diseño orientado a la experiencia de usuario.  

> ⚠️ Nota: gran parte del CSS contiene código repetido, ya que en esta materia no se evaluaba el CSS, sino HTML, JavaScript y manejo de Figma.

---

## 🛠️ Tecnologías utilizadas
- HTML5, CSS3, JavaScript (ES6+)  
- Librerías según corresponda (por ejemplo, Bootstrap o Swiper.js para carousels)  
- Git para control de versiones  

---

## ⚙️ Instalación
1. 📥 Clonar el repositorio:
```bash
git clone https://github.com/fernandocordeschi/interfaces.git
```

2. 📁 Entrar a la carpeta del proyecto:
```bash
cd TP-Interfaces
```

3. 🌐 Abrir el index.html en tu navegador.
